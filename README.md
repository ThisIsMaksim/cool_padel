# Cool Padel

Flutter-приложение для ведения счёта матчей по паделу и теннису. Работает на **Android**, **iOS**, **Web**, **Windows**, **Linux** и **macOS**.

## Режимы

### Стандартный
- Матч до 2 или 3 сетов
- Геймы до 6 с разницей в 2
- Тай-брейк при счёте 6:6
- Очки в розыгрыше: 0 → 15 → 30 → 40, deuce и advantage

### Турнирный
- Задаётся суммарное количество очков в матче
- 1 выигранный мяч = 1 очко
- Матч завершается, когда сумма очков обеих команд достигает лимита

## Запуск

```bash
# Web
flutter run -d chrome

# Android / iOS / desktop
flutter run
```

## Тесты

flutter test
